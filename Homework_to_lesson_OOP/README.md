# Homework_to_lesson_OOP

### 1. To run tests use commands in "test" directory:

- All tests:

`pytest -s`

- Only shape class tests

`pytest -sm shape`

- Only square class tests

`pytest -sm square`

- Only rectangle class tests

`pytest -sm rectangle`

- Only triangle class tests

`pytest -sm triangle`

- Only circle class tests

`pytest -sm circle`
